### Описание
**PhotoFilter** - это сайт, на который пользователи могут загружать фотографии, редактировать их и делиться ими. Все фото сохраняются на сайте, а их владельцы могут устанавливать права доступа к ним. 

### Зависимости
+ Ruby 3.2.2
+ Node.js
+ PostgreSQL
+ Tailwind
